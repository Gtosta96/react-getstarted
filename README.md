# Tecnologias #

 - Yarn
 - Bower
 - Gulp
 - React 
 - Webpack
 - Babel
 - Eslint

**Execução do projeto**
```
$ npm install yarn bower gulp
$ yarn cache clean & yarn
$ bower cache clean & bower install

$ npm run dev
```

**Utils**
```
$ eslint app/**/* 
    Verifica arquivos em busca de problemas de formatação baseado nas regras do Eslint.
```
